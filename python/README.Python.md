# Media files manager


## Installation

Common Python installation
```python
pip install -r requirements.txt
```

Using Python Launcher
```python
py -m pip install -r requirements.txt
```


## Running
```python
py media-files-manager.py
```

| Argument  | Comment      |
|-----------|--------------|
| arg       | Path         |
| -h        | Help         |
| --draft   | Draft mode   |
| --execute | Writing mode |

Example command could be:
```python
py media-files-manager.py /home/user/media/photos --execute
```
